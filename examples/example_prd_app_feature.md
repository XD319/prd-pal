# PRD — 社交 App "附近的人" 功能 v1.0

## 背景

产品数据显示，当前 App 用户日均使用时长为 18 分钟，其中 70% 集中在消息聊天模块。为提升用户发现效率和社交活跃度，计划上线"附近的人"功能，允许用户基于地理位置发现周边活跃用户并发起互动。竞品分析表明，同类功能可使新用户次日留存率提升 8-12 个百分点。

## 功能列表

### F-1 位置发现

- 用户开启定位后，展示半径 1km / 3km / 5km 内最近活跃的用户卡片列表。
- 卡片信息：头像、昵称、距离（精确到百米）、个性签名、最近活跃时间。
- 列表按距离升序排列，支持下拉刷新和无限滚动加载。

### F-2 隐私控制

- 用户可随时关闭"附近的人"功能，关闭后自身不出现在他人列表中。
- 提供"隐身模式"：可浏览他人但自身不可见。
- 支持黑名单：被拉黑用户不会出现在对方的附近列表中。

### F-3 互动入口

- 点击用户卡片进入个人主页，可查看动态、共同好友数。
- 提供"打招呼"快捷操作，发送预设文案或自定义短消息（≤ 50 字）。
- 每人每天打招呼上限 20 次，超出后提示次日刷新。

### F-4 安全策略

- 未成年用户（< 18 岁）不展示在"附近的人"列表中，也无法使用该功能。
- 打招呼消息经内容安全审核后投递，包含违禁词的消息直接拦截。
- 对频繁被举报的用户自动降低曝光权重。

## 非功能需求

- **性能**：附近用户列表接口响应 ≤ 500ms（P99），支持 10 万级并发地理查询。
- **定位精度**：使用设备 GPS + Wi-Fi 混合定位，精度 ≤ 100 米。
- **电量优化**：后台定位更新间隔 ≥ 5 分钟，避免持续 GPS 导致电量消耗 > 5%/小时。
- **合规**：位置数据存储遵循《个人信息保护法》，用户删除账号时同步清除位置记录。
